# Bulk Watermark Removal

**Bulk Watermark Removal** is a desktop application that removes watermarks from images
using OpenCV inpainting. It supports both preset watermark positions and custom
freehand masking with a live preview.

- Freehand watermark masking with adjustable brush size
- Live preview with overlay (preset + freehand)
- Drag & drop image support
- Progress bar with live ETA
- Summary report after processing
- Stable, professional UI layout
